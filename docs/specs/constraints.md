# 차량 커뮤니티
프로젝트 명세서 작성
##	1. imprint 사용자 가입 ##
	- 식별번호(id)
	- 이메일(e-mail) 
	- 비밀번호(password) 
	- 별명(nickname) 
	- 이름(name) 
	- 나이(age)
- 제약 조건
    - 별명, 이메일은 각각 중복될 수 없음(중복 가입 여부 확인 목적)
    - ID는 중복될 수 없다.
- 기능
    - 가입 후 개인정보 열람가능
    - 회원 정보 수정가능
    - 계정 분실 시 이메일 인증을 통해 계정 찾기 가능
    - 회원가입 후 잃어버린 계정을 찾으려면 이메일 인증을 통하여 찾아야한다.

## 2. 게시판 ##
	- 게시판 번호(id)
	- 전체 게시판 (free board)
	- 게시판 생성 시간 (created_at)

- 기능
    - 각 게시판은 고유한 게시판 번호(id)로 식별된다.
    - 게시판에는 여러 개의 게시물이 등록될 수 있다.
    - 관리자는 게시판을 추가or삭제할 수 있다.

## 3. 게시물 ##
	- 게시판 번호(id)
 	- 게시물 번호(board-id)
	- 제목 (title)
	- [댓글개수][첨부파일] 표시
	- 작성자 (ueser-id)
	- 내용 (content)
	- 조회수 (hits)
	- 추천수 (recommend)
	- 시간(수정(update_at) / 작성(created_at))

- 게시물은 게시물 번호(id)로 식별된다.
- 사용자는 제목과 내용을 입력하여 새로운 게시물을 작성할 수 있다.
- 게시물에는 여러 개의 댓글과 첨부파일이 연결될 수 있다.

## 4. 첨부파일(attachments) ##
	- 첨부파일 번호(id)
	- 댓글 번호(id)
	- 게시판 번호 (board_id)
	- 게시물 번호 (post_id)
	- 작성자 ID (user_id)
	- 파일명 (file name)
	- 파일 경로 (path)
	- 파일 크기 (size)
	- 작성시간 (created_at)

- 첨부 파일은 고유 번호(id)로 식별된다.
- 하나의 게시물에는 여러 개의 첨부 파일이 등록 될 수 있다.
- 사용자는 게시물 작성시 첨부 파일을 추가 및 삭제 가능하다.
- 첨부파일은 게시물과 관계를 맺으며 필요시 댓글에도 확장 가능하다.

## 5. 댓글 ##
	- 댓글 번호(id)
	- 댓글 작성자 (board-id)
	- 내용 (title)
	- 추천수 (recommend)
	- 시간(수정(update_at) / 작성(created_at))

- 댓글은 댓글 번호(id)로 식별된다.
- 사용자는 게시물에 댓글을 작성할 수 있으며, 여러 개의 댓글을 남길 수 있다.

## 6. 쪽지 ##
	- 쪽지 번호(id)
	- ID 참조 (송신자 ID [sender_id] / 수신자 ID [receiver_id])
	- 내용 (content)
	- 발송 시간 (sent_at)
	- 쪽지확인 여부 (is_read)

- 쪽지는 송신자와 수신자의 사용자 ID를 참조한다.
